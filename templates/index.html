This server is a demonstration Datasette instance. It showcases the following Datasette plugins:

  • datasette-ui-extras, for power-user browsing and editing UI
  • datasette-parquet, for serving Parquet, CSV, JSON and DuckDB files
  • datasette-current-actor, for auditing changes to database tables

Anyone may log in with a GitHub account. Logged-in users can edit tables in the cooking database.
